# 🌾 Krishi Wise Loan - Frontend

Frontend for Krishi Wise Loan application built with React + Vite.

---

## 🚀 Features
- User Login & Registration UI
- Dashboard with EMI analysis
- Government schemes listing
- Weather widget
- Multi-language support (Hindi + English)
- Responsive UI

---

## 🛠 Tech Stack
- React.js
- Vite
- TypeScript / JavaScript
- Tailwind CSS
- Axios

---

## 📦 Installation

```bash
cd krishi-wise-loan-main
npm install

⚙️ Environment Variables
Create .env file:
VITE_API_URL="https://backend-1-xj9l.onrender.com/"
▶️ Run Project
npm run dev

App runs at:
http://localhost:5173

🔗 Backend Connection

Make sure backend is running on:

http://localhost:5000
